# 01. Clasificación Binaria

Se parte del articulo de Higham & Higham para la resolución de clasificación de pozos petroleros, se sigue el pseudocódigo y se replican los mismos resultados, despues evoluciona el código para poder configurar la red y sus hiperparámetros, para finalmente agregar mejoras en el código como inicialización de pesos y sesgos, el optimizador Adam, y condiciones de paro adicionales.
